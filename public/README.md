# Public Assets

Place your static assets here, such as images.

## Profile Photo

To add your profile photo:
1. Add your photo file to this `public` folder
2. Name it `profile-photo.jpg` (or update the path in `src/components/About.jsx`)
3. Supported formats: JPG, PNG, WebP
4. Recommended size: 800x800px or larger (square aspect ratio works best)

If no photo is found, the website will display your initials "HM" as a fallback.
